








### Adding to Github from workspace
1. git init
2. git add . {for all file}
3. git commit -m "message"
4. git branch -M main
5. git remote add origin https://github.com/Anurag-chandan/repository_name.git
6. git push -u origin main

### Pushing existing repository from command-line
1. git remote add origin https://github.com/Anurag-chandan/repository_name.git
2. git branch -M main
3. git push -u origin main
